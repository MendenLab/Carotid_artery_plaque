# Instructions to reproduce main figures of the manuscript

1. scRNA-seq:
   - Run all Jupyter Notebooks in the ```scrna_analysis/reproducing_main_figures``` folder. Before running them, replace the ```result_dir``` variable in all cells of all Notebooks with your desired path of directory to collect the results into.


2. Xenium:
   - Run all Jupyter Notebooks in the ```xenium_analysis/reproducing_main_figures``` folder. Before running them, replace the ```result_dir``` variable in all cells of all Notebooks with your desired path of directory to collect the results into.
  

3. Bulk RNA-seq:
   - Run all R files in the ```xenium_analysis/Codes_BulkRNAseq``` folder. Before running them, replace the ```output_dir``` variable in all .R files with your desired path of directory to collect the results into.
